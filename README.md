# mknpm
mknpm is a tool to generate a npm package that has bin directory quickly. You can use this tool when you want to publish a shell command.

### Requirements
* macOS or Ubuntu
* Node.js
* npm
* git

### Usage
**install**
```bash
npm install -g TakutoYoshikai/mknpm
```

**generate a npm package**
```bash
mknpm <COMMAND NAME>
```
